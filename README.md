# cloudmark-api

A Simple bookmark CRUD API.

## Getting Started

1. git clone https://github.com/the-redlord/cloudmark-api.git
2. npm install
4. npm index.js

### Prerequisites

* NodeJS
* npm

## Deployment

Deploy to your server to access the API.

* Postgresql
